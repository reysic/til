# Creating Aliases

Recommended to create aliases in `$ZSH_CUSTOM`, to edit:

```bash
nano $ZSH_CUSTOM/aliases.zsh
```

Sample alias:

```text
alias k="kubectl"
```

Following any changes, run:

```bash
omz reload
```

To list aliases:

```bash
alias
```
